# @deessejs/collections

A functional-first collection and data modeling layer built on Drizzle ORM.

## Features

- **Functional First**: Composable field types and collections
- **Type Safe**: Full TypeScript inference from Zod schemas
- **Drizzle Native**: Seamless integration with Drizzle ORM
- **Plugin System**: Extensible with native and custom plugins
- **i18n Ready**: Built-in internationalization support
